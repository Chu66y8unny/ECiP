Interactive Python <ipython.org>

```shell
$ # If using Conda package manager we should already have IPython
$ conda update ipython # update to the most recent version
$ # If Python is installed
$ pip install ipython
$ # If using Ubuntu
$ sudo apt-get install ipython 
```
